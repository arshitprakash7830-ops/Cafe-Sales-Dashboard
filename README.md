# Cafe-Sales-Dashboard
Interactive Excel Cafe Sales Dashboard tracking 73K+ transactions and 24K+ items sold. Analyzes revenue drivers (Salads/Sandwiches), temporal traffic peaks (Sundays/Fridays), and location performance (In-store vs. Takeaway). Includes payment method splits and multi-criteria slicers (Day/Month) to optimize inventory and staffing. in Google lokker studio
